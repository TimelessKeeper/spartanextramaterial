# Desctiption

A Minecraft mod to add the missing material from [Materials Expanded](https://www.curseforge.com/minecraft/mc-mods/matex) to [Spartan Weaponry](https://www.curseforge.com/minecraft/mc-mods/spartan-weaponry).
